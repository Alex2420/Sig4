Conclusion: 

Ce cours a permis de montrer tout d'abord les différentes inégalités dans la santé entre les hommes et les femmes. Mais aussi de nombreuses inégalités entre les femmes à n'importe quel âge. Les jeunes filles n'ont pas toutes les mêmes facilités à accéder aux moyens de contraception. Cela à des repercussions sur leur éducation mais aussi leur santé. Les femmes n'ont pas les mêmes soins durant leur grossesse ce qui provoque de nombreuses maladies. Ces inégalités perdurent dans les maladies comme les cancers. Les inégalités ne cesse de continuer et de plus elles sont présentes dans les études, dans les entreprises, dans l'éducation, les differences de salaires... 

A retenir: 

Ces inégalités entre les femmes ne doivent pas continuer et dans notre pays nous avons l'accès nécessaire pour une bonne santé et il ne faut pas l'oublier. Nous avons le droit à l'avortement qui n'est pas le cas dans tous les pays du monde.
